개인정보처리방침

본 개인정보처리방침은 '하나 봇'(이하 "서비스")이 Discord 사용자 정보를 어떻게 처리하는지에 대해 설명합니다.

1. 수집하는 정보

서비스는 Discord API를 통해 사용자 ID, 서버 ID, 메시지 내용 등을 수집할 수 있으나, 저장 여부는 기능에 따라 달라집니다.

2. 정보의 사용

수집된 정보는 서비스 기능 제공, 통계 분석, 문제 해결에 사용됩니다. 마케팅이나 외부 공유에는 사용되지 않습니다.

3. 보관 기간

개인정보는 사용 목적 달성 후 지체 없이 삭제되며, 별도 저장소에 보관되지 않습니다.

4. 제3자 제공

서비스는 어떠한 개인정보도 제3자에게 제공하지 않습니다.

5. 문의

개인정보 관련 문의는 GitHub 또는 Discord 서버를 통해 문의할 수 있습니다.
